# TokTik Clone - Flutter (Solo Mensajes)

TokTik es un clon de la interfaz de usuario de TikTok enfocado en la visualización de videos cortos y dinámicos, simulando la experiencia de scroll vertical. Este proyecto utiliza datos estáticos para demostrar la interfaz de usuario, gestos y animaciones sin necesidad de un backend.

## 🚀 Características

- **UI Similar a TikTok:** Scroll vertical de pantalla completa.
- **Datos Estáticos:** Datos simulados listos para visualizar sin utilizar un BackEnd.
- **Scroll de videos:** Simulación de scroll continuo.
- **Interfaz Animada:** Transiciones suaves.


## 🛠️ Tecnologías Utilizadas

- **Flutter**: Framework de UI multiplataforma.
- **Dart**: Lenguaje de programación.
- **PageView**: Widget principal para el scroll vertical.
- **Gestión de Estado**: (`setState`, `Provider`)
- **Video Player**

## 💻 Cómo Ejecutar el Proyecto

### Requisitos Previos

- [Flutter SDK](https://docs.flutter.dev) instalado.
- [Visual Studio Code](https://code.visualstudio.com) o Android Studio.

### Pasos

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/lsantiagosc/flutter-tok-tik.git
    ```
2.  **Entrar en la carpeta:**
    ```bash
    cd toktik-clone
    ```
3.  **Instalar dependencias:**
    ```bash
    flutter pub get
    ```
4.  **Ejecutar la app:**
    ```bash
    flutter run
    ```
